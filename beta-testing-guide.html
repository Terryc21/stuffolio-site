Beta Testing Guide

Thank you for helping test Stuffolio! Your feedback shapes the app before public release.

Current Beta: Version 1.0 (Build 27) ‚Äî Updated February 24, 2026

Getting Started

Install via TestFlight

  1. Install Apple's TestFlight app on your iPhone, iPad, or Mac.
  2. Tap the Stuffolio beta invite link.
  3. Install Stuffolio and start testing!

First Steps

  • Add a few items ‚Äî Start with 3-5 items you know well (electronics, appliances, tools)
  • Try the new Add Item flow ‚Äî Choose Photo (AI-assisted), Barcode, or Manual entry
  • Enable iCloud sync (optional) ‚Äî Test syncing between devices if you have multiple Apple devices
  • Explore AI assistance ‚Äî Try Stuff Scout, AI Product Assistant, and the new Current Price lookup (runs only when you ask)

Beta Tester Benefit: You have unlimited AI access during the beta period. Use Stuff Scout, Photo analysis, and AI Assistant as much as you want ‚Äî no query limits while testing!

Important: Beta builds expire after 90 days. TestFlight will notify you when updates are available. Always keep the app updated to the latest beta build.

What to Test

Focus your testing on these priority areas:

AI assistance when you ask ‚Äî silent when you don‚Äôt.

Stuffolio includes AI assistance that runs only when you explicitly request it. There is no background analysis, no passive monitoring, and no automatic data sharing.

High Priority

  • Item Deletion ‚Äî Delete items with photos and verify no crashes
  • iCloud Sync ‚Äî Sync between devices and check for duplicate items
  • Repair Advisor ‚Äî Try with damaged items, check recommendations make sense
  • Go To (⌘K) ‚Äî Search for features like "Settings" or "Value Calculator" and verify they open directly
  • Crashes ‚Äî Report any app crashes immediately

Medium Priority

  • AppleCare+ Editing ‚Äî Try editing AppleCare+ details directly from the dashboard
  • Icon Colors ‚Äî Check that icons use distinct, meaningful colors (not all blue)
  • Photo Loading ‚Äî Scroll through items and verify photos load smoothly
  • AI assistance ‚Äî Stuff Scout accuracy, AI Assistant responses
  • Price Watch ‚Äî Track prices, set alerts, view history charts
  • Export functionality ‚Äî CSV, PDF, Excel, encrypted backup exports

Lower Priority

  • Empty State ‚Äî Check the iCloud Sync option appears when inventory is empty
  • UI polish ‚Äî Visual glitches, alignment issues
  • Accessibility ‚Äî VoiceOver, Dynamic Type
  • Widgets ‚Äî Home Screen, Lock Screen widgets

New in Build 27

This build focuses on reliability, accessibility, and making features easier to reach.

Colorblind-Friendly Icons

  • Icons now use distinct colors instead of the previous "all blue" look
  • Categories, conditions, and statuses are easier to tell apart
  • Check icons in item lists, the Dashboard, and category views

Smarter Repair Advisor

  • Open Repair Advisor from the Dashboard or an item's context menu
  • If you have one damaged item, it opens the advisor directly (no picker step)
  • Recommendations now use your actual item data ‚Äî age, warranty status, condition, and your history with that brand
  • Look for confidence levels: High, Medium, or Exploratory

Direct AppleCare+ Editing

  • Open AppleCare+ Tracking from the Dashboard
  • If you have one Apple product, it opens the editor directly
  • Edit incidents used, service fees, and coverage dates without opening the full item editor

Go To Opens Features Directly

  • Press ⌘K or tap the search icon to open Go To
  • Search for "Value Calculator" ‚Äî it should open Value Calculator directly
  • Search for "Settings" ‚Äî it should open Settings directly
  • Search for "Depreciation" ‚Äî it should open the Depreciation tracker directly
  • Previously these would land on a general page where you'd have to find the feature yourself

Crash Fixes

  • Try deleting items that have photos attached ‚Äî this was crashing before
  • Try deleting items that were synced from another device
  • Check that gallery view shows photos correctly after syncing

iCloud Sync Improvements

  • Add an item on one device and check it appears on another without duplicates
  • Verify photos sync correctly between devices
  • When your inventory is empty, check that an "Enable iCloud Sync" option appears

Still Worth Testing from Earlier Builds

  • Encrypted Backups: Settings ‚Üí Backup ‚Üí toggle "Encrypt with password"
  • Send to Apps: Tap Notes, Calendar, or Reminders chips on any item
  • Scan Depth Options: Try Quick ID, With Pricing, and Full Appraisal in Stuff Scout
  • Data Tools: Data Cleanup, Batch Categorize, Bulk Price Update

Testing Checklist

Use this checklist to ensure comprehensive coverage:

Core Functionality

  • Add items using all 3 methods (Photo, Barcode, Manual)
  • Test Photo mode ‚Äî verify form appears immediately while AI loads
  • Verify AI fields fill in automatically after photo analysis
  • Edit item details and verify changes save
  • Delete an item and confirm it's removed
  • Delete an item with photos attached ‚Äî verify no crash
  • Search for items by name, category, location
  • Filter items using the filter menu
  • Export inventory to CSV, PDF, or Excel

Backup & Security

  • Create a plain (unencrypted) backup and verify restore
  • Create an encrypted backup with password
  • Verify encrypted file has .stuffolio extension
  • Restore encrypted backup ‚Äî verify password prompt appears
  • Try wrong password ‚Äî verify error message
  • Restore with correct password ‚Äî verify data integrity

Family Sharing

  • Create a shared household
  • Invite a family member as Editor
  • As Editor, try to delete an item you didn't create
  • Verify deletion is blocked with clear message
  • As Admin, verify you can delete any shared item
  • Verify item creator can always delete their own items

Price Watch

  • Enable Price Watch on an item
  • Add a price manually
  • Set a price alert with target price
  • View price history chart
  • Use AI "Current Price" to fetch prices
  • Apply Stuff Scout value to Price Watch

AI Features

  • Use Stuff Scout to identify an item from photo
  • Test Scan Depth: Quick ID (~15s)
  • Test Scan Depth: With Pricing (~25s)
  • Test Scan Depth: Full Appraisal (~40s)
  • Refine Scout results with additional details
  • Use AI Product Assistant for manual/parts lookup
  • Try Current Price query
  • Verify AI response caching works offline

Data Tools

  • Run Data Cleanup analysis on your inventory
  • Check quality scores for different categories
  • Use Batch Categorize on uncategorized items
  • Verify keyword matching suggestions
  • Use Bulk Price Update with percentage adjustment
  • Use Bulk Price Update with AI lookup
  • Test Photo Optimizer on large photos
  • Verify optimized photos maintain quality

Legacy Wishes

  • Enable Legacy Wishes in Settings
  • Add at least 2 beneficiaries
  • Assign items to beneficiaries
  • Set contingency recipients
  • Test sharing via CloudKit (if family available)
  • Export Legacy Wishes as PDF

Repair Advisor & AppleCare+

  • Open Repair Advisor from Dashboard ‚Äî check recommendations use your real item data
  • If you have one damaged item, verify it skips the picker and opens directly
  • Look for confidence levels (High, Medium, Exploratory) on recommendations
  • Open AppleCare+ Tracking from Dashboard
  • If you have one Apple product, verify it opens the editor directly
  • Edit AppleCare+ incidents, fees, and dates from the direct editor

Go To (⌘K)

  • Open Go To and search for "Value Calculator" ‚Äî verify it opens directly
  • Search for "Settings" ‚Äî verify it opens Settings directly
  • Search for "Depreciation" ‚Äî verify it opens the tracker directly
  • Search for "Cloud Sync" ‚Äî verify it opens sync settings directly
  • Navigate to a section (like My Products or Dashboard) via Go To

Sync & Multi-Device

  • Enable iCloud sync
  • Add item on one device, verify it appears on another
  • Verify photos sync correctly between devices
  • Check for duplicates after syncing
  • Edit same item on two devices, test conflict resolution
  • Test with airplane mode / offline usage
  • When inventory is empty, check that an "Enable iCloud Sync" option appears

How to Report Feedback

Primary: Use TestFlight's built-in feedback ‚Äî shake your device or go to TestFlight ‚Üí Stuffolio ‚Üí Send Beta Feedback

Email: beta@stuffolio.app ‚Äî For detailed reports, screenshots, or feature requests

  • What you were doing ‚Äî Steps to reproduce the issue
  • What you expected ‚Äî The behavior you expected
  • What actually happened ‚Äî The actual result (crash, error, wrong data, etc.)
  • Device info ‚Äî iPhone/iPad/Mac model, iOS/macOS version
  • Screenshots/screen recordings ‚Äî Visual evidence is extremely helpful
