Version 1.0 (Build 25)

February 10, 2026 Beta

Critical fixes for App Store readiness. Improved CloudKit sync reliability, better error recovery, and enhanced diagnostics. Plus simplified onboarding, enhanced family sharing safety, and optional encrypted backups.

Critical Fix

CloudKit Batch Sync ‚Äî Replaced sequential record uploads with batch operations for better performance and reliability. Better error recovery continues on non-critical errors instead of stopping completely.

Highlight

Simplified Onboarding ‚Äî New "Set Up Your Experience" screen replaces the previous privacy consent wall. Three simple toggles (AI Assistants, Online Lookups, iCloud Sync) are all ON by default with clear explanations of what each does.

Security

Password-Protected Backups ‚Äî Optionally encrypt your backup files with a password. Encrypted backups use industry-standard PBKDF2 + AES-GCM encryption and are saved as .stuffolio files. Restore automatically detects encrypted files and prompts for the password.

New Features

  • Set Up Your Experience ‚Äî Action-oriented onboarding with 3 feature toggles (AI, Online Lookups, iCloud Sync) ‚Äî all enabled by default
  • Encrypted Backups ‚Äî Optional password protection for backup exports using AES-GCM encryption
  • Family Sharing Deletion Safety ‚Äî Only the item creator or household Admin can delete shared items ‚Äî Editors can no longer accidentally delete others' items
  • Scan Depth Options ‚Äî Choose your analysis level: Quick ID (~15s), With Pricing (~25s), or Full Appraisal (~40s) ‚Äî balance speed vs detail for your needs
  • Data Cleanup Tool ‚Äî Analyze inventory completeness with quality scores, identify items missing photos, values, or warranties, and get suggestions for improving data quality
  • Batch Categorize Tool ‚Äî Automatically categorize multiple items at once using keyword matching ‚Äî great for cleaning up uncategorized imports
  • Bulk Price Update Tool ‚Äî Update prices for multiple items using various strategies: percentage adjustment, AI lookup, or manual entry
  • Photo Optimizer ‚Äî Reduce photo storage size while maintaining quality ‚Äî compress existing photos or set defaults for new captures

Improvements

  • Consistent Terminology ‚Äî Changed "Product" to "Item" throughout the app for clearer, more intuitive language
  • Onboarding Hints ‚Äî Each toggle now includes subtle explanatory text about what data is used
  • Delete Permission Feedback ‚Äî Clear messaging when users can't delete shared items they didn't create

Critical Fixes (February 10)

  • CloudKit Batch Sync ‚Äî Batch operations replace sequential uploads; processes items in groups of 200
  • Partial Deletion Handling ‚Äî Failed deletions remain in tracker for retry instead of being lost
  • Enhanced Error Logging ‚Äî Sentry captures all CloudKit errors for better diagnostics
  • App Store Compliance ‚Äî Removed exit(0) patterns that could cause rejection; replaced with state-driven recovery

Under the Hood

  • Separate Test Schemes ‚Äî Unit tests and UI tests now run separately for faster development
  • Refactored Large Files ‚Äî Dashboard, Item Detail, and Form views split into focused extension files for better maintainability
  • Mock Managers ‚Äî New protocol-based mocks for CloudSync, Backup, and Keychain testing

Price Watch ‚Äî See How Inflation Affects What You Own ‚Äî That mixer you paid $350 for might cost $550 to replace today.   • Conflict Resolution ‚Äî When two people edit the same item, compare versions and choose what to keep
  • Offline Support for Shared Items ‚Äî Edit shared items offline; changes sync automatically when back online
  • Price Watch ‚Äî Track current replacement costs for any item. Enable per-item to monitor prices from multiple sources
  • Price History Charts ‚Äî Visual timeline showing how prices change over weeks or months
  • Price Drop Alerts ‚Äî Set target prices and get notified when items drop below your threshold
  • AI "Current Price" Query ‚Äî New AI Assistant query finds current retail prices from Amazon, Walmart, Best Buy, Target, and more
  • Stuff Scout ‚Üí Price Watch ‚Äî When applying Scout results, optionally update Price Watch with marketplace values
  • Barcode ‚Üí Price Watch ‚Äî UPC lookup now returns current retail prices; option to auto-enable Price Watch
  • Dashboard Price Watch Card ‚Äî See items with significant price changes at a glance
  • Smart Auto-Fill ‚Äî AI discovers manufacturer websites and support page URLs during product research. Save them to your product with one tap. A new Resources section appears in item details only when URLs are populated (progressive disclosure)
  • Component-Based Warranties ‚Äî Track different warranty periods for different product components (e.g., 3 years on batteries, 5 years on tools). Category-aware suggestions help you quickly add common components
  • Coverage Phases Discoverability ‚Äî Coverage Phases is now a top-level section in the add/edit form (no longer nested inside Warranty). Available when creating new items, not just editing
  • Coverage Phase Help ‚Äî New in-form help button explains coverage types, component warranties, and when to use phases
  • Legacy Wishes CloudKit Sharing ‚Äî Share your Legacy Wishes with family via iCloud with real-time sync across all devices
  • Legacy Wishes Family Input ‚Äî Family members can indicate preferences on items: "I'd like this", "No preference", or "I don't need this"
  • Legacy Wishes Conflict Detection ‚Äî When multiple family members express interest in the same item, the owner sees a conflict indicator to help start conversations
  • Legacy Wishes Offline Support ‚Äî View and work with shared Legacy Wishes even when offline; changes sync automatically when back online

  • AI Response Caching ‚Äî Price lookups cached for offline access; reduced API usage
